# 'Simple' Implementation of Parallel Computation and Distributed System of a Social Media Platform

With focus on uploading file using multithreading, database replication, communication between node, and Redis queue.

# ⚙️ Requirements
- Python
- Redis server _(I'm using Memurai Redis for Windows, but Docker Redis also supported)_
- Python libraries -> flask, requests, redis
- A lot of lucks _(For real. Somehow my PC and laptop has different behavior of Memurai Redis)_
